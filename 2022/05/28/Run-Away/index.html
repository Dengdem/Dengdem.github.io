Run away

by Black Holic

爱是﻿﻿﻿﻿run away?

有一次很晚了他都没有走，可能是白天又要拍戏又要写歌实在太累了。我鬼使神差地提议，不如留一晚好了。没想到他居然说好的，他躺在我的胳臂上，整个人蜷缩在我的怀抱中，用一种自然的口吻。我忍不住替他提了提被子，以免他着凉鼻炎又发作。实际上屋子里暖气很足，露一只手臂在外一整晚也不会感冒。但我还是那样做了，好像我们是相处了很久的，真正意义上的couple。

我并不是一个善于做记录与规划的人，比起记住在哪天做了什么有趣的事，我更喜欢记住那一瞬间的感觉，然后重复，再积攒，最后变成一种生理上难以抹去的记忆。所以我并不能清楚地回忆起具体是哪一天，或许是他房间空调坏了的那天，应该是了。那天还下了雪，拍摄中止了。搭车﻿回宾馆的路上他看起来很疲惫。我能体会，创作的瓶颈，多重的压力。我安抚地捏了捏他的手。﻿

“还好吗宰燦？”﻿

他看向我时又恢复了亮晶晶的眼神。“是，哥，我还好。”﻿

他叫我哥。片场外也叫。几乎不叫我栖含哥，只是单字“哥”。和剧里的尚宇一样。我有时恍惚也以为自己是张宰英了，但我不会动画制作，简笔画倒是不错，可对电脑使用一窍不通。但他也不像秋尚宇。他和我一样数学不好，也不爱做计划。想到这儿，我反而能坦然接受他叫我哥了，不论是以尚宇的口吻，还是从宰燦的角度。﻿

刚开机那几天我常失眠。不全是在思考如何演绎张宰英这个闪闪发光的角色，虽然这点也确实很让我困扰。对于如何定义自己我有着长久的困惑，这种困惑随着退出组合而不断地加深。如果倒退十年我会不会更能驾驭这个角色？每晚睡前这个问题总是不合时宜地跳进我的脑袋。睁着眼睛想不出，闭上眼睛思绪更是一团乱。这极大地影响了我的睡眠质量，加上刚开始我还是自驾往返片场与租的公寓，休息时间进一步被压缩，第二天顶着两只黑眼圈开工，抱歉地对化妆师说请给我调一盘无敌的遮暇吧！﻿

他好像敏锐地发现了我的问题所在。在一个我正要驱车在午夜前赶回住所的夜晚，他借着给我递暖宝宝拽住了我的袖子。﻿

“哥。”他带着有两只耳朵的毛线帽，卸了妆，羽绒服的领子遮住了半张脸，看上去像准备艺考的高中生。“和我住一个宾馆吧，离这不远，晚上还能一起回去。”﻿

在这种情形下很难说不，而且不得不说这确实是一个非常贴心的建议。于是我没有犹豫就答应了，从此我睡在他的隔壁房间。﻿

酒店的设施很齐全也很新，只有一点，隔音不是太好。有时我能听见他烦恼地来回踱步，有时是合成器的声音，极少时是吉他。﻿

难以平静时我趴在自带的枕头上翻阅剧本。还是老生常谈的自我怀疑，翻来翻去编剧把张宰英很有魅力这几个字刻画在字里行间。到底为什么找我演呢？不禁又想起导演这两天对我的评价，说我有些太束手束脚了。我觉得她说得对，张宰英应该是很张扬肆意的。但我很想问，宰英才二十二岁，我已经三十岁了诶！我都已经快要忘记自信两个字要怎么写了。将剧本掩在脸上，模糊间快要睡着时，有人敲响了我的门。﻿

我猜我此刻的样子一定很憔悴，对上眼时他明显愣了一下，但马上笑开。﻿

“哥，我好冷。”﻿

我这才注意到他只穿了一件有破洞的白色毛衣。﻿

“那快进来吧。”下意识地用毯子将他裹住，他的身体很薄。今天下午有一场拥抱戏，我抱住他，像抱住一片有温度的叶子，或者说一片有温度的叶子刚好飘进了我的怀中。﻿

“我房间的空调坏了。”﻿

“我帮你打电话让前台来修吧。”我匆忙地从箱子中拿出姜茶，想给他泡一杯。﻿

“已经打过了，说是师傅这周末才会来，会退我房费的。”﻿

他乖乖坐在那里的样子很可爱，让人忍不住想好好照顾。﻿

“哥，我不喝姜茶。”﻿

“噢？好吧，我想让你暖和一点。”﻿

“谢谢哥。”他环顾四周，晃了晃脚，“或许有酒吗？”﻿

其实我并没有随身带酒的习惯，但居然在行李箱的角落里挖出半瓶威士忌。他一副我早就知道的表情，我不知为何就开始解释，“可能是上次朋友的庆生宴上落下的，随手塞进去了。”我自己灌了一杯，又给他倒上一杯时突然意识到他还是现役偶像，开始真切地担心。“宰燦你没关系吗？我以为你只喝牛奶呢。”他好像听到了什么天大的笑话一样笑得很开心。“哥，你真的很可爱你知道吗？”可能是我已提前一杯酒下肚的缘故，也有可能是我们凑得太近，我甚至能看到他颤动的睫毛，和浅浅的内双。我很恶趣味地去捏他的脸，“小孩子还说别人可爱吗？”他大概以为是自己用了平语我才假装生气，所以又贴近我一点说，“对不起冒犯含含哥了。”﻿

含含是个很稀罕的称呼。我从小最不缺的就是名字，最开始家长说名字很重要，关乎一个人的气运。后来经纪公司说名字很重要，星途是否坦荡就看名字响不响亮。虽然很多人曾为了我的名字绞尽脑汁，却很少有人给我起昵称，要么干脆叫后两个字，礼貌又亲切。我本来也不知道朴栖含这个名字又能坚持多久，但我突然有点喜欢它了。﻿

我让他闭眼，然后将酒杯轻轻贴在他的眼睛上，像melo的体质中孙锡久对李恩静做的那样。他说“哥，好凉。”然后我将酒杯换成了自己的唇。﻿

第二天我们依旧搭乘同一辆车去片场。化妆师开玩笑地问我们昨晚是不是结伴干了什么坏事，怎么黑眼圈的浓度都如此相似。我和他对视一笑，默契地没有将昨晚的欲仙欲死说出口。“可能是失眠症会传染吧。”我说。“没错。”他附和。﻿

之后偶然变成了常规。白天我们在片场接吻，晚上又在我房间的床上接吻。他高潮时也喜欢喊我“哥”。于是我有时分不清剧里剧外，分不清在我床上高潮的是偶像兼演员朴宰燦还是书里的计算机系学生秋尚宇。也有感到确认的时刻，比如有时做完了感到肚饿出门买宵夜，他拿着鱼饼对我说“含含哥张嘴。”又比如他总在片场连名带姓地夸我，“朴栖含是我见过最棒的演员。”“栖含老师，可以教我一下这里怎么酝酿情绪吗？”“朴栖含太过分了，演得这么好，演他的对手戏真苦恼啊！”﻿

我习以为常地请他吃饭，有次他找了很多次借口去上洗手间，却每次都往反方向走，等吃完我才意识到他是想要买单。我说“谢谢宰燦呐”，他回“含含哥请了太多次，内心真的会很不好意思。”我又开始自作自受地发散思维，如果是张宰英请秋尚宇吃饭，尚宇会感到不好意思地回请吗。我知道这是不理智的无用功，但我总是恍惚，恍惚导演喊cut却好像真正的戏才刚刚开始，恍惚还没开机张宰英却已经住进了自己的身体。我不懂什么体验派、理论派之分，我只是个演过一次爱情剧演技麻麻，截至目前一事无成的退役偶像。我不是闪着耀眼光芒的张宰英，但我已经迷上当他的感觉了，至少他有秋尚宇。

空调修好后他来我房间的频率也还是很高。不做时我们会躺在一起看电影，美其名曰精进演技。准备看《爱在日落黄昏时》（这片子当然是他的口味，但偶尔看看不合口味的电影也无妨。）我不禁向他推荐我家里购置的日落灯，“和这片名很搭吧？”他笑着点点头说确实很搭，然后问我“要不干脆留到周末到哥家里看好了？”这回轮到我愣住，但也只愣了两秒，我听见自己说“好啊。”﻿

﻿

事情变得有些不可控制。我有时怀疑难道他也享受出不了戏的状态吗？不然为什么会周末了也想和工作对象一起度过呢？伴着片尾曲，我们第一次在公寓的床上做了。他好像很喜欢我的被子，说是触感很好还问了我好几次洗衣剂的牌子。天色渐亮，我关上日落灯，阳光透过百叶窗打在他的侧脸，我突然觉得被一种从未有过的莫名情绪填充了。他将另外半张脸埋在我的被子里说“好喜欢哥的味道，好想把哥的味道装进瓶子里带走。”我没有忍住，于是我们又做了一次，他仍旧将脸埋在被子里我从后面进入。应该换个遮光窗帘我突然想到，如果还会有很多个这样的清晨。﻿

对于即将到来的离别我强装镇定。我口是心非地用他分享给我的歌曲的歌词开解他，对话框里我打下：离别是romance。后半句我没有打，爱是run away吗？我不这么认为。﻿

我渐渐觉得他是体验派，分不清戏里戏外。为了自我保护，我开始逐渐抽离，但收效甚微。于是我只好言语抗议，我去他文案为“哥的工作室”的ins下留言生气表情。我开始感到失望和愤怒，对他也对自己。真的没办法认清我是谁吗？我为什么要奢求现实中的朴宰燦也爱我呢？即使是作为尚宇对宰英的爱也够了不是吗？好好接着就是了，不管爱来自于谁。﻿

他的悲伤随着杀青日的临近而如放入石块的水杯，日益满溢。我以为他是难舍片场乌托邦般的生活，也难舍作为拥有很多爱的尚宇。有次我看他难过实在不忍，做到情动时我紧紧抱住他说“不要难过，就当哥是宰英好了，宰英会一直在的。”但他却流泪了，这让我手足无措。我吻去他的泪水，心中苦涩又卑微。我想宰燦啊，千万不要发现我真的爱上你了，不是作为张宰英，不是爱上秋尚宇。﻿﻿

时间无情地碾过，最后一天还是来临。他穿着心爱的紫色卫衣，相拥而泣时我闻到了熟悉的香味。﻿

“哥..你的…同款洗衣剂…真的很好用…清洁力很强…还很香”他抽泣着断断续续说出这样的话，我听得心都快化了。我边不争气地抹眼泪边说：好用哥再给你买。他却摇了摇头。“哥..宿舍真的很小…已经放不下..第二瓶…想用你家的…可以吗？”我反应了好一阵才明白他话里的意思。﻿﻿

“不是尚宇喜欢吗？”我知道我的眼睛又红了。﻿

“不是的。是宰燦喜欢。”﻿

“请与我交往吧！”我埋在他的颈间轻声说。然后我听见他说“好”。﻿
